In this release, we've added **Winget** (Windows Package Manager) support for uninstalling and upgrading our OpenTelemetry Collector on Windows using install scripts. This improves deployment flexibility, streamlines upgrades on supported systems, and maintains compatibility with existing installation methods. [Learn more](/docs/send-data/opentelemetry-collector/install-collector/windows/#uninstall).
