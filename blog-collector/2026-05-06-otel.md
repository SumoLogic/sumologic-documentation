We're excited to introduce the **Domain Controller Mode** for Windows Active Directory (AD) event collection in OpenTelemetry Collectors. By setting `discover_domain_controllers: true` in your custom YAML configuration, the collector automatically discovers all domain controllers on the AD network, connects to each controller's event log, and begins monitoring events. [Learn more](/docs/send-data/opentelemetry-collector/remote-management/source-templates/customyaml/#configure-domain-controller-mode-for-windows-ad-event-collection).
