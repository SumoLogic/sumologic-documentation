This content release includes:
* Expanded Ubiquiti Unifi coverage with 7 new log mappers for process execution monitoring, DHCP events, DNS queries, and network traffic visibility.
* Enhanced Abnormal Security threat detection with additional email metadata and user identification fields for improved threat investigation.
* Improved Netskope WebTx field mappings for network correlation including source/destination IPs, ports, and file hash tracking (MD5 and SHA-256).

Additional changes are enumerated below.
