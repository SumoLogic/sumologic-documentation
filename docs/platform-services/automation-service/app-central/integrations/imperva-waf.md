Imperva WAF offers web application security firewall, providing protection against the most sophisticated security threats. As a cloud-based WAF, it ensures that your website is always protected against any type of application layer hacking attempt.

## Actions

* **List Sites** *(Enrichment)* **-** Retrieves a list of all sites.


## Imperva WAF in Automation Service and Cloud SOAR


1. To configure the integration, log into the application, expand the configuration menu in the top right corner by hovering over the gear icon and click Automation. <br/>![](/img/platform-services/automation-service/app-central/integrations/imperva-waf/imperva-waf-1.png)

1. In the Automation section, on the left menu, click Integrations. <br/>![](/img/platform-services/automation-service/app-central/integrations/imperva-waf/imperva-waf-2.png)

1. After the list of the integrations appears, search for the integration and click on the row.

1. The integration details will appear. Click on the "+" button to add new Resource. <br/>![](/img/platform-services/automation-service/app-central/integrations/imperva-waf/imperva-waf-3.png)

1. Populate all the required fields (\*) and then click Save.
   * Label: the name for the resource
   * URL API: 'https://my.imperva.com'/
   * API ID
   * API Key <br/>![](/img/platform-services/automation-service/app-central/integrations/imperva-waf/imperva-waf-4.png)

1. To make sure the resource is working, hover over the resource and then click the pencil icon that appears on the right. <br/>![](/img/platform-services/automation-service/app-central/integrations/imperva-waf/imperva-waf-5.png)

1. Click Test Saved Settings. <br/>![](/img/platform-services/automation-service/app-central/integrations/imperva-waf/imperva-waf-6.png)   
 
1. You should receive a successful notification in the bottom right corner. <br/>![](/img/platform-services/automation-service/app-central/integrations/imperva-waf/imperva-waf-7.png)
