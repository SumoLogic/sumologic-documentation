**Nmap** is a free and open-source network scanner. Nmap is used to discover hosts and services on a computer network by sending packets and analyzing the responses.

## Actions

* **Create Scan** (*Enrichment*) - Start NMAP scan
* **Check Scan Status** (*Enrichment*) - Check scan status
* **Get Scan Info** (*Enrichment*) - Get scan info
* **Get Scan Report** (*Enrichment*) - Get scan report
* **List Credits** (*Enrichment*) - Get API calls report for current member

## Nmap in Automation Service and Cloud SOAR

1. To configure the integration, log into the application, expand the configuration menu in the top right corner by hovering over the gear icon and click Automation. <br/>![](/img/platform-services/automation-service/app-central/integrations/nmap/nmap-1.png)

1. In the Automation section, on the left menu, click Integrations. <br/>![](/img/platform-services/automation-service/app-central/integrations/nmap/nmap-2.png)

1. After the list of the integrations appears, search/look for the integration and click on the row.

1. The integration details will appear. Click on the "+" button to add new Resource. <br/>![](/img/platform-services/automation-service/app-central/integrations/nmap/nmap-3.png)

1. Populate all the required fields (\*)
   * URL: Your URL default ('https://api.nmap.online'/)
   * Api Key : Insert the API Key

1. Click Save. <br/>![](/img/platform-services/automation-service/app-central/integrations/nmap/nmap-4.png)

1. To make sure the resource is working, hover over the resource and then click the pencil icon that appears on the right. <br/>![](/img/platform-services/automation-service/app-central/integrations/nmap/nmap-5.png)

1. Click Test Saved Settings. <br/>![](/img/platform-services/automation-service/app-central/integrations/nmap/nmap-6.png)

1. You should receive a successful notification in the bottom right corner. <br/>![](/img/platform-services/automation-service/app-central/integrations/nmap/nmap-7.png)
