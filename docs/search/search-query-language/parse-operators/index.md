# Parse Operators

Parse operators allow you to extract fields from log messages within a query manually and on an ad-hoc basis.

For best practices use Parse operators to build [Field Extraction Rules] (../../Manage/Field-Extractions.md) to automatically extract field values and use them to extend your query.
