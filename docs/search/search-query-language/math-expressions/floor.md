The floor function rounds down to the largest previous integer value. Returns the largest integer not greater than x.

## Syntax

```sql
floor(<x>) as <field>
```

## Example

```sql
floor(1.5) as v // v = 1
```

```sql
floor(-1.5) as v // v = -2
```
