### Enhancements 

We're excited to announce the release of the enhancements listed below for the Sumo Logic apps:

- **Added Monitors in OpenTelemetry apps**. Sumologic provides six new monitors for Apache app with a number of alerts that are triggered on different conditions. [Learn more](/docs/integrations/web-servers/opentelemetry/apache-opentelemetry/#apache-alerts).
- **Added Unified Audit log in OpenTelemetry apps**. Added the ability to gather Unified Audit logs using Syslog and Windowsevent receivers. Additionally, this dashboard offers information on database users, top current users, and trends in logon status. This dashboard can also be used with the unified audit logs exported from both Windows and Linux environments. [Learn more](/docs/integrations/databases/opentelemetry/oracle-opentelemetry/#unified-audit-syslog).
- **Added CloudTrail dashboard for AWS Application Load Balancer**. The CloudTrail Audit dashboard visualizes successful and failed events globally, event trends, error details, and user activities, offering insights into load balancer performance, security, and usage patterns. You can receive a comprehensive overview of the [AWS Application Load Balancer](/docs/integrations/amazon-aws/application-load-balancer/#cloudtrail-audit), [AWS Classic Load Balancer](/docs/integrations/amazon-aws/classic-load-balancer/#cloudtrail-audit), and [AWS Network Load Balancer](/docs/integrations/amazon-aws/network-load-balancer/#cloudtrail-audit), and their activities through the CloudTrail audit logs.
- **Amazon RDS**. Added dashboards of Oracle DB for CloudWatch Logs, CloudWatch Metrics, and CloudTrail logs. [Learn more](/docs/integrations/amazon-aws/rds/#oracle-logs---alert-logs-analysis).
