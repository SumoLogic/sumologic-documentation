We're excited to announce the release of our new cloud-to-cloud source for Trust Login. 

Trust Login helps you to collect the Report Logs from the Trust Login API and send them to Sumologic. Trust Login enhances trust, convenience, and robust security measures, revolutionizing the authentication process and ensuring reliable access control without compromising user experience. [Learn more](/docs/send-data/hosted-collectors/cloud-to-cloud-integration-framework/trust-login-source).
