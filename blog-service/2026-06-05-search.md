We're excited to annunce that you can now copy a JSON value name or the entire JSON structure directly from your search results by clicking the **Copy** icon next to an array or JSON value, or by highlighting the text, right-clicking, and selecting **Copy field value**. [Learn more](/docs/search/get-started-with-search/search-basics/view-search-results-json-logs/#copy-message-content).
