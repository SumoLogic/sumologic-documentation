### Enhancements

We’ve enhanced multiple Sumo Logic apps to improve visibility, usability, and proactive monitoring with refreshed queries, localized dashboards, and new monitor alerts.

Added new dashboards, monitor alerts, and an IDP category for the following apps:
- [Active Directory 2012+ - OpenTelemetry](/docs/integrations/microsoft-azure/opentelemetry/active-directory-json-opentelemetry/#viewing-active-directory-dashboards). Removed the deprecated batch processor from the OpenTelemetry configuration. Batching is now enabled by default with the Sumo Logic exporter. The updated OpenTelemetry configuration is compatible with Sumo Logic OpenTelemetry Agent version 0.153 and later.
- [Active Directory 2012+ (JSON)](/docs/integrations/microsoft-azure/active-directory-json/)
- [Auth0](/docs/integrations/saml/auth0/#viewing-the-auth0-dashboards)
- [Azure Active Directory](/docs/integrations/microsoft-azure/active-directory-azure/#viewing-azure-active-directory-dashboards)
- [ChatGPT Compliance](/docs/integrations/saas-cloud/chatgpt-compliance/#viewing-the-chatgpt-compliance-dashboards). Updated dashboard queries to align with the latest log format, and added a `custom_gpt` selection variable to the dashboard for easier filtering.
- [JumpCloud Directory Insights](/docs/integrations/saas-cloud/jumpcloud-directory-insights/#viewing-the-jumpcloud-directory-insights-dashboards)
- [OneLogin](/docs/integrations/saml/onelogin/#viewing-onelogin-dashboards)
- Migrated the following Classic Apps to Next-Gen Apps:
    - [Amazon DynamoDB](/docs/integrations/amazon-aws/dynamodb/)
    - [Amazon ECS without Container Insights and Traces](/docs/integrations/amazon-aws/elastic-container-service)
    - [Amazon ElastiCache](/docs/integrations/amazon-aws/elasticache/)
    - [Amazon RDS](/docs/integrations/amazon-aws/rds)
    - [Amazon SNS](/docs/integrations/amazon-aws/sns/)
    - [Amazon SQS](/docs/integrations/amazon-aws/sqs/)
    - [AWS Application Load Balancer](/docs/integrations/amazon-aws/application-load-balancer/)
    - [AWS API Gateway](/docs/integrations/amazon-aws/api-gateway/)
    - [AWS Classic Load Balancer](/docs/integrations/amazon-aws/classic-load-balancer/)
    - [AWS EC2 CW Metrics](/docs/integrations/amazon-aws/ec2-cloudwatch-metrics/)
    - [AWS EC2 Host Metrics](/docs/integrations/amazon-aws/ec2-host-metrics/)
    - [AWS Lambda](/docs/integrations/amazon-aws/lambda/)
    - [AWS Network Load Balancer](/docs/integrations/amazon-aws/network-load-balancer/)

#### Collection Mechanism

- **AWS Lambda Extension v1.4.1**. Released the security patch addressing Denial of Service vulnerabilities in the AWS SDK EventStream Decoder (CVE fixes for `s3` and `eventstream` protocol). Updated AWS SDK Go v2 dependencies to the latest stable versions. [Learn more](https://github.com/SumoLogic/sumologic-lambda-extensions/releases/tag/v1.4.1).
- **VMware Tanzu Integration v1.0.9**. Upgraded Golang to v1.26.4 and updated dependencies to address 7 CVEs in `golang.org/x/net`. Updated deprecated `HttpStart`/`HttpStop` event types. Now compatible with VMware Tanzu Application Service 10.4.0 Small Footprint Elastic Application Runtime. [Learn more](https://github.com/SumoLogic/sumologic-cloudfoundry-nozzle/releases/tag/v1.0.9).
