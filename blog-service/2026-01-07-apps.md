### New release

#### LiteLLM

We are excited to announce that we have added a native Sumo Logic HTTP Source webhook integration for collecting LiteLLM usage and proxy data in Sumo Logic.

### Enhancements

- **VMware Tanzu Application Service**. Released Sumo Logic Nozzle for VMware Tanzu versions 1.0.7 and 1.0.8, which add support for Tanzu v10.2. For detailed version information, refer to the [changelog file](https://github.com/SumoLogic/sumologic-cloudfoundry-nozzle/blob/master/CHANGELOG.md).
- **AWS Serverless Application Models (SAM)**. Released the following two SAMs:
    - `sumologic-aws-cloudtrail-benchmark 1.0.22`
    - `sumologic-guardduty-benchmark 1.0.19`
