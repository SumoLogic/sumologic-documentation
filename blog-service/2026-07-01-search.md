Mobot introduces higher-level reasoning capabilities that go beyond simple query translation. This unified conversational platform automatically analyzes log data behind the scenes, correlates information across sources, and provides structured answers to both data analysis and platform how-to questions. The enhanced interface includes thinking indicators, follow-up question suggestions, and improved contextual understanding to accelerate troubleshooting and reduce mean time to detection and response (MTDR).

**What's new:**
* Unified conversational interface - no need to select agents or modes
* Higher-level reasoning with implicit log analysis and multi-step planning
* Thinking and planning indicators show analysis in progress
* Follow-up question suggestions guide investigation workflow
* 30-day conversation history retention
* Automatic routing between log analysis and knowledge base questions
* Enhanced contextual understanding using RAG and dashboard context
