We're excited to introduce the new Sumo Logic app for Netskope WebTx Streaming. This app collects web transaction logs streamed from the Netskope platform via an Amazon S3 source, providing security and IT teams with real-time visibility into web transactions to monitor, analyze, and secure their web traffic. [Learn more](/docs/integrations/saas-cloud/netskope-webtx-streaming).
