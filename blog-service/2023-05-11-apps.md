We're excited to announce the release of our new Zoom source for hosted collector. This source provides an endpoint for receiving webhook events that help you to monitor:
- Meeting events
- Webinar events
- Recording events
- Zoom Room events
- User events
- Account events 

For more information, check out our technical documentation [here](/docs/send-data/hosted-collectors/webhook-sources/zoom/).
