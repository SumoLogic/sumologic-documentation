Mobot introduces higher-level reasoning capabilities that go beyond simple query translation. This unified conversational interface automatically analyzes log data behind the scenes, correlates information across sources, and provides structured answers to both data analysis and platform how-to questions. The enhanced interface includes thinking indicators, follow-up question suggestions, and improved contextual understanding to accelerate troubleshooting and reduce mean time to resolution (MTTR).

**What's new:**
* Unified conversational interface with no agent or mode selection required.
* Higher-level reasoning with implicit log analysis and multi-step planning.
* Thinking and planning indicators show analysis in progress.
* Follow-up question suggestions guide your investigation workflow.
* Extended conversation history retention.
* Automatic routing between log analysis and platform how-to questions.
* Enhanced contextual understanding using dashboard context.
