### New Release

We’re excited to announce the release of the following apps for Sumo Logic.

- **Amazon SageMaker**. The Amazon SageMaker app provide insights into CloudTrail, CloudWatch Logs, and performance metrics for your Amazon SageMaker service. These preconfigured dashboards help you monitor logs and track runtime performance efficiently. [Learn more](/docs/integrations/amazon-aws/amazon-sagemaker/).
- **Azure Firewall**. The Azure Firewall app enables centralized visibility into firewall health, network and application rules, threat intelligence, and IDPS events. It helps you monitor, analyze, and respond to security threats across your Azure workloads with greater efficiency and scalability. [Learn more](/docs/integrations/microsoft-azure/azure-firewall/).
- **Azure Subscription**. The Azure Subscription app centralizes the collection and analysis of activity logs, enhancing security, operational visibility, and compliance monitoring. It provides actionable insights across your Azure environment by organizing logs within the subscription boundary for efficient monitoring and governance. [Learn more](/docs/integrations/microsoft-azure/azure-subscription/).
- **Data Volume for MSSP**. The Data Volume for MSSP app provides centralized visibility into data usage across child organizations by type, tier, category, and source. With predefined dashboards and searches, it helps optimize data ingestion, manage costs, and improve operational oversight. [Learn more](/docs/integrations/sumo-apps/data-volume-mssp/).
- **Apache Hadoop - OpenTelemetry**. The Apache Hadoop - OpenTelemetry app delivers end-to-end visibility into cluster health, performance, and resource utilization through logs and metrics. With preconfigured dashboards for HDFS and YARN, it enables real-time monitoring, faster troubleshooting, and efficient capacity planning. [Learn more](/docs/integrations/bigdata/opentelemetry/apache-hadoop-opentelemetry/).
- **OpenLLMetry**. The OpenLLMetry app enables end-to-end observability of LLM applications by collecting traces, metrics, and events across prompts, responses, latency, and errors. It provides deep insights into interactions between models, APIs, and vector databases, helping optimize performance and accelerate troubleshooting. [Learn more](/docs/integrations/aiml/opentelemetry/openllmetry/).

### Enhancements

We’ve enhanced multiple Sumo Logic apps to improve visibility, usability, and proactive monitoring with refreshed queries, localized dashboards, and new monitor alerts.

- **AWS Lambda Extension**. The Sumo Logic AWS Lambda Extension from arn version 12 (release [v1.4.0](https://github.com/SumoLogic/sumologic-lambda-extensions/releases/tag/v1.4.0)) now supports Lambda managed instance runtime and adds deployment support for two new AWS regions: ca-west-1 and eusc-de-east-1. This update enhances runtime compatibility and expands regional availability, enabling more flexible and scalable deployments. [Learn more](/docs/send-data/collect-from-other-data-sources/collect-aws-lambda-logs-extension/#sumologic-lambda-extension-aws-region-availability).
