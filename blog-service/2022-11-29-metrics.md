We've updated the logic Sumo Logic uses to calculate input data limits for metrics queries. The new method will enable upcoming improvements: higher cardinality limits for time series inputs to metric queries. For more information, see [Metrics Query Error Messages](https://help.sumologic.com/docs/metrics/metrics-queries/metric-query-error-messages/#input-data-limit).
